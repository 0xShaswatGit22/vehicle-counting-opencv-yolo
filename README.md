# 🚗 Vehicle Counting using YOLOv8 & OpenCV

A simple **Computer Vision project** that detects and counts vehicles in a video using **YOLOv8** and **OpenCV**.

---

## 📌 Features

✅ Real-time vehicle detection
✅ Bounding box around detected vehicles
✅ Count of vehicles displayed on screen
✅ Works on recorded traffic videos
✅ Built using **Python + YOLOv8 + OpenCV**

---

## 🧠 Technologies Used

* 🐍 Python
* 👁️ OpenCV
* 🤖 YOLOv8 (Ultralytics)
* 🔢 NumPy

---

## 📂 Project Structure

```
vehicle-counting-opencv-yolo
│
├── main3.py          # Main vehicle detection & counting script
├── video.mp4         # Input traffic video
└── README.md
```

---

## ⚙️ Installation

Install required libraries:

```bash
pip install ultralytics opencv-python numpy
```

---

## ▶️ Run the Project

```bash
python main3.py
```

The program will:

1️⃣ Read the video
2️⃣ Detect vehicles using YOLOv8
3️⃣ Draw bounding boxes
4️⃣ Display vehicle count on screen

---

## 🚦 Supported Vehicle Classes

🚗 Car
🚌 Bus
🚚 Truck
🏍️ Motorcycle

---

## 📸 Output Example

* Vehicles detected with **green bounding boxes**
* Vehicle **count displayed on screen**

Example:

```
Cars: 5
Trucks: 2
Bus: 1
Motorcycle: 3
```
